Multikernel Cloud

无虚拟化开销的云性能

面向云工作负载的分离内核架构。每个应用程序都拥有自己的内核，配备专用核心和原生硬件访问权限。无需虚拟机管理程序，没有嘈杂邻居，没有性能损耗。可在 AWS、GCP 或 Azure 上任意标准云虚拟机内运行。

快速上手 预约演示

您将获得什么

从经过验证的内核到生产级编排，运行云工作负载并达到裸金属性能所需的一切。

经过验证的内核构建

预构建、经过测试的内核配置，针对云工作负载优化。针对低延迟、高吞吐量调优，并已在 AWS、GCP 和 Azure 实例类型上完成验证。

Kerf 编排

通过单一工具创建、配置和管理内核实例。内置资源分配、生命周期管理、监控与策略执行。

DAXFS 共享内存

连接应用内核与设备内核的高速共享内存文件系统。零拷贝数据传输、共享容器根文件系统，并与 Docker 集成。

生产级工具链

监控、告警，并与现有基础设施集成。兼容 Kubernetes、Terraform 以及标准 Linux 管理工作流。

优势

零嘈杂邻居

每个工作负载都运行在专用核心上属于自己的内核中。没有中断干扰，没有 CPU 争用，也没有共享内核开销窃取您的运算周期。

裸金属性能

没有虚拟机管理程序损耗，没有嵌套虚拟化。应用程序直接访问硬件，弥合了云虚拟机与裸金属之间的性能差距。

独立生命周期

更新或替换任意内核实例，不影响其他实例。每个内核都拥有自己的更新路径和故障域。

随处运行

裸金属、AWS、GCP、Azure，均可运行。无需特殊实例类型。在任何标准云虚拟机内实现内核级隔离。

兼容 Docker

直接启动进 Docker 镜像。无需修改即可使用您现有的容器、镜像和工作流。

100% 开源

构建在上游 Linux 之上。所有组件均在 GPL-2.0 协议下开源。没有专有锁定。
