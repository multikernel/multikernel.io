Multikernel Private Cloud

无需虚拟机管理程序的私有云

在您自主掌控的基础设施上整合 Linux 工作负载。每个工作负载都拥有自己的内核，配备专用核心和原生硬件访问权限——无论是您数据中心里的裸金属，还是您已租用的虚拟机。无需虚拟机管理程序，没有嘈杂邻居，没有性能损耗。

快速上手 预约演示

您将获得什么

从经过验证的内核到生产级编排，以裸金属性能运行私有云所需的一切。

经过验证的内核构建

签名的内核、Kerf 与 DAXFS 软件包（rpm/deb），通过私有软件仓库交付。每个版本一套受支持的配置，支持 Secure Boot，并已在公开的硬件兼容性列表（覆盖服务器硬件与云实例类型）上完成验证。

Kerf 编排

通过单一工具创建、配置和管理内核实例。内置资源分配、生命周期管理、监控与策略执行。

DAXFS 共享内存

连接应用内核与设备内核的高速共享内存文件系统。零拷贝数据传输、共享容器根文件系统，并与 Docker 集成。

生产级工具链

监控、告警，并与现有基础设施集成。兼容 Kubernetes、Terraform 以及标准 Linux 管理工作流。

优势

零嘈杂邻居

每个工作负载都运行在自己的内核中，独占专用核心。没有中断干扰，没有 CPU 争用，也没有共享内核开销窃取您的运算周期。

裸金属性能

没有虚拟机管理程序损耗，没有虚拟化堆栈。应用程序直接访问硬件，整合后的工作负载依然像独占整机一样快。

独立生命周期

更新或替换任意内核实例，不影响其他实例。每个内核都拥有自己的更新路径和故障域。

您的基础设施，随处部署

无论是您数据中心里的裸金属，还是您已租用的虚拟机。无需特殊实例类型，无需新购硬件。在您自主掌控的基础设施上实现内核级分区。

兼容 Docker

直接启动进 Docker 镜像。无需修改即可使用您现有的容器、镜像和工作流。

100% 开源

构建在上游 Linux 之上。所有组件均在 GPL-2.0 协议下开源。没有专有锁定。

如何获取

Multikernel Private Cloud 是一款开源软件，您可以将其安装在自己的基础设施上。免费下载、自行构建；企业订阅按节点按年计价。

开源版

下载并从源码构建。完整访问 multikernel 内核、Kerf 编排和 DAXFS。通过 GitHub issues 获得社区支持。
